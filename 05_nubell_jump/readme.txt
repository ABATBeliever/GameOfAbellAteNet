1. ゲーム概要
=============================
ダブルジャンプできるぬーべる君を操作して山を登りましょう！
単純な仕組みだけど、これまでのゲームよりサクサク遊べる！

=============================
2. インストール方法
=============================
Web版:
ページを開きます。

PWA版 - PC(Chrome/Edge):
...(三点リーダ)>アプリ>ぬーべるジャンプ！ のインストール

PWA版 - Android(Chrome):
メニュー>アプリをインストール>インストール
※選択の場合は「ショートカット」ではなく「PWA」を選ぶ

PWA版 - iOS(Safari):
共有アイコン>ホーム画面に追加>追加
※選択の場合は「ショートカット」ではなく「PWA」を選ぶ

=============================
3. 遊び方
=============================
PCの場合:
矢印キーで左右移動、Spaceでジャンプ

スマホ・タブレットの場合:
スマホを傾けて左右移動、タップでジャンプ
※スマホでもキーボード操作は可能です

5. 問題が発生した場合
=============================
問題が発生した場合、以下の手順を試してください:
・ブラウザのキャッシュをクリアし、ページを再読み込みしてください。
・ゲームの最新版をダウンロードしてインストールし直してください。
・問題が解決しない場合は、サポートにお問い合わせください:
