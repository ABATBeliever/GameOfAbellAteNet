=============================
1. ゲーム概要
=============================
無心ドロップスは、PC、スマホ、タブレット対応の王道の落ちものパズルゲームです。
キャラを集めてスコアを競い合いましょう。
シンプルな操作で、初心者から上級者まで楽しめます。

=============================
2. インストール方法
=============================
Windows版:
1. ダウンロードしたファイルをダブルクリックして起動します。インストールはありません。
※Windows Defenderにご注意ください

ビルドする場合:
hsp.tvよりHSP開発環境を入手し、スクリプトエディタで編集し、コンバータでdishにできます。
公式では、HSP3.6を想定しています。

=============================
3. 遊び方
=============================
1. ゲームを開始すると、キャラが上から降ってきます。
2. キャラを触って二人以上集めてポイントを集めます。
3. 操作は指又はマウスでどうぞ
4. 最高スコアを目指して挑戦しましょう！

=============================
5. 問題が発生した場合
=============================
問題が発生した場合、以下の手順を試してください:
・ブラウザのキャッシュをクリアし、ページを再読み込みしてください。
・ゲームの最新版をダウンロードしてインストールし直してください。
・問題が解決しない場合は、サポートにお問い合わせください:
   >> https://x.com/abatbeliever

6. ライセンス
=============================
無心ドロップスのソースコードはMITライセンスの下で提供されています。
ただし、依存関係であるHSP3.6のOBAQにはBSDが別途課せられているため、ビルド時にはご注意ください。
